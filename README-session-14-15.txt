GitFusion v22 Session 14-15

Correções desta versão:
- Sidebar visualmente mais próxima do ChatGPT: lista limpa, texto menor e sem caixas gigantes.
- Projetos não aparecem automaticamente em Recentes: só aparecem depois de criar/gerar um projeto.
- Cada projeto agora tem uma área própria chamada "Chats" dentro dele.
- Recentes pode ficar vazio, mostrando apenas "Sem recentes".
- Itens dentro das pastas ficaram menores e mais simples.
- Mantido o gesto apenas Chat <-> Sidebar, sem Preview no arrasto.

Instalação Termux:
cd ~/GitFusion
unzip -o /sdcard/Download/GitFusion-v22-session-14-15-chatgpt-sidebar-project-chats.zip
npm install
npm start
