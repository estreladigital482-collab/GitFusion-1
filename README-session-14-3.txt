GitFusion v22 Session 14.3 - Project Tree + Tasks + Sidebar por pastas

Inclui:
- Novo projeto cria uma pasta de projeto na sidebar.
- Cada pasta pode conter vários chats.
- Setinhas para abrir/fechar pastas e árvore.
- Recentes aparecem primeiro, com limite visual na sidebar.
- Área de Tasks estilo Tasklight.
- Repositórios mesclados abrem em árvore tipo VS Code.
- GitHub e Configurações removidos da sidebar principal.
- GitHub permanece em Perfil > Configurações > Publicar no GitHub.
