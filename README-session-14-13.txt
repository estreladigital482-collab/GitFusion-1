GitFusion v22 - Session 14.13

Fix de gesto em 3 páginas:
- A navegação mobile agora respeita três posições: Sidebar, Chat e Preview.
- O Chat é o ponto central obrigatório.
- Arrastar da esquerda para a direita a partir do Chat abre somente a Sidebar.
- Fechar a Sidebar volta e para no Chat.
- O Preview só abre com um gesto separado da direita para a esquerda a partir do Chat, ou pelo botão Play.
- A Sidebar continua preenchendo 100% da tela no mobile.
