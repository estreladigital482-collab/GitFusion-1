# Sessão 9: Terminal flutuante

- Terminal abre pelo botão `+` do composer.
- Janela flutuante dentro da área principal/chat.
- Pode mover, fechar, minimizar, expandir e limpar.
- Antes de cada comando, o usuário precisa autorizar.
- Backend executa no workspace ativo com bloqueios básicos para comandos perigosos.
- Saída stdout/stderr aparece na janela do terminal.

Próxima evolução: permitir a IA sugerir comandos como cards autorizáveis no chat.
